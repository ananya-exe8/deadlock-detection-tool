# Automated Deadlock Detection Tool

A Python-based Operating Systems project that detects deadlocks using graph algorithms and visualizes process-resource relationships.

## Features
- Deadlock detection
- Near deadlock warning
- GUI dashboard
- Resource Allocation Graph
- Resolution suggestions
- Simulation mode
- Export reports

## Tech Stack
- Python
- PyQt6
- NetworkX
- Matplotlib
- psutil

## Run

pip install -r requirements.txt
python gui.py
